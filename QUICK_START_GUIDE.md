## 🚀 Deployment Guide

### Quick Deploy to Streamlit Cloud (5 min)

**Step 1: Setup SSH (one-time)**
```bash
ssh-keygen -t ed25519 -C "your-email@gmail.com"
# Add to GitHub Settings → SSH & GPG Keys
```

**Step 2: Commit Code**
```bash
cd c:\Users\hp\Desktop\NLPpro
git add -A
git commit -m "Resume Screening Pro v4.1 - Premium UI"
git push origin main
```

**Step 3: Deploy**
1. Go to https://share.streamlit.io
2. Sign in with GitHub
3. Click "New App"
4. Select repository: `gaurav2302221-cell/NLP-G1`
5. Branch: `main`
6. File: `resume_screening_project/app.py`
7. Click "Deploy"
8. Wait 2-3 minutes
9. 🎉 You're live!

### Your App URL
```
https://resume-screening-[unique-id].streamlit.app
```
