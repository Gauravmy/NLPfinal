PIPELINE CAPABILITIES
================================================================================

RANKING MODELS SUPPORTED:
  1. TF-IDF Vector Similarity
     • Keyword-based matching
     • Fast computation
     • Good for traditional resumes

  2. BERT Embeddings
     • Semantic understanding
     • Captures meaning beyond keywords
     • Better for complex requirements

  3. Hybrid Approach
     • Combines TF-IDF (70%) + Skill Matching (30%)
     • Balanced approach
     • Suitable for skill-focused roles

  4. Deep Ensemble
     • Combines all methods with weights:
       - TF-IDF: 50% (keyword matching)
       - BERT: 30% (semantic understanding)
       - Skills: 20% (skill matching)
     • Best overall performance
     • Robust ranking system

EVALUATION METRICS:
  • Precision@K - Percentage of relevant items in top-K
  • Recall@K - Percentage of relevant items retrieved in top-K
  • Mean Average Precision (MAP) - Ranking quality metric
  • Model comparison visualizations

FEATURES:
  ✓ Automatic skill extraction from resumes
  ✓ Job requirement parsing
  ✓ Candidate ranking by similarity
  ✓ Multiple model comparison
  ✓ Performance evaluation
  ✓ Result visualization and export
  ✓ CSV and image output
